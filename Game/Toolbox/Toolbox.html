Toolbox

*Some of these tools can work together, but note that they were designed to be exclusive.
Do not feel like you must use a tool if it can interact with other tools.

Worlds

The Worlds tool allows GMs to build large scale maps, describing generally what their game's landscape looks like.
It could be a fantasy world with continents and oceans with differing planes of existence and a complex underdark.
Or it could be a deep Sci-Fi universe detailing each solar system within a galaxy.

Maps

The Maps tool allows GMs to detail specific environments, whether it be the Tomb of Horrors or The Starship Enterprise. Note that maps are drawn in a hex grid format.
If you so desire, you can also use the maps tool to reference places created in the Worlds tool.

Encounters

The Encounters tool allows GMs to prep enemies for a game.
When the GM is done preparing they can whip up a few fast enemies whenever needed.
This tool also includes some pre-made enemies that might look familiar...

Planner

The Planner tool allows GMs to prepare for contingencies in a game.
It helps GMs plan for multiple possibilities when players make decisions.
It also helps GMs plan events for when players enter a town or a specific zone/area.

History

The History tool allows GMs to detail the history of their world.
The history tool lets GMs take notes on places or events that have happened.
This tool can be used with the Worlds tool to connect historical data to places in the world.

Networks

The Networks tool allows GMs to create organizations such as social, political, cultural, or religious groups.
The Networks tool also has a variety of details you can use to characterize a network. Such as the socio-economic status of the group, positions within a hierarchy, or even regular contact information.
